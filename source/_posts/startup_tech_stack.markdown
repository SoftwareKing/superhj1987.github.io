# 初创公司技术栈

- apiGateway：统一api网关
- api doc：api说明文档
- db： 数据库
- search：搜索引擎
- redis：nosql/缓存
- esb：服务治理框架
- app center：统一app中心
- log system：日志系统
- fault monitor：故障监控
- file storage：文件存储
- schedule center：调度中心
- rpc：rpc调用机制
- app profile: 性能能监控分析
- distribute config center:统一配置中心

## 基础设施
- 代码版本管理：Gitbucket
- Java代码管理：Maven
- 自动化部署+code检查：Jenkins + checkstyle + findbugs + pmd + sonar
- 项目管理：禅道
